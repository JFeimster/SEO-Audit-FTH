Welcome to the SEO Audit and Action Plan repository for **[Full-Time Hustler](https://fthustler.wordpress.com/)**.

## Overview
This repository contains the findings, structured data templates, and prioritized action plan generated by the Claude Code SEO Audit Suite for the Full-Time Hustler website. The audit identifies critical legal/trust risks, significant technical SEO gaps, and high-impact opportunities for AI search visibility (GEO).

- `/` - Root documentation, audit reports, and action plans.

## Getting Started
To begin improving the site's SEO, start with the **Critical** items in the Action Plan or the checklist in `TASKS.md`. The highest priority is adding the FTC affiliate disclosure to all posts.
