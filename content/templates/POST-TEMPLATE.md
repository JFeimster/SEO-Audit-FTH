# Standard SEO Post Template

## Pre-Publish Checklist
- [ ] FTC Affiliate disclosure added at the very top of the post body
- [ ] Affiliate links carry `rel="nofollow sponsored"` attributes
- [ ] Title tag under 60 characters
- [ ] Custom meta description between 140–155 characters
- [ ] Alt text added to all images
- [ ] Minimum word count of 1,500 words
- [ ] At least 2–3 named and hyperlinked external sources/citations
- [ ] 3–5 contextual internal links with descriptive anchor text
- [ ] 3–5 Q&A structured FAQ section at the end of the post
- [ ] BlogPosting schema JSON-LD block injected
- [ ] Post broken into 5–7 named sections with H2 headings
- [ ] Author bio correctly linked to About page

## Post Structure

### 1. The Hook (Introduction)
*Briefly introduce the topic, identify the problem the reader faces, and outline how this post will solve it.*

### 2. Main Body (H2 Headings)
*Break content into logical sections using H2 and H3 headings. Incorporate data points, external links, and internal links.*
*If using AI-assisted drafting, ensure sections are fully fleshed out with original insights and personal specifics.*

### 3. Practical Steps / Tutorials
*Include actionable steps, tool walkthroughs, or case studies where applicable.*

### 4. Conclusion
*Summarize the main takeaways and provide a call to action.*

### 5. Frequently Asked Questions (FAQ)
*Add 3-5 Q&A pairs directly addressing user queries to capture AI Overviews and rich snippets.*
- **Q:** [Insert Question]
  **A:** [Direct, self-contained 50-120 word answer without hedging]
